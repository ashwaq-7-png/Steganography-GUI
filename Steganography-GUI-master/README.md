# **Features 🌱**
1. **GUI Interface:** A user-friendly graphical interface built with Tkinter for easy interaction.

2. **Data Hiding:** Embed text data into an image file, making it invisible to the naked eye.

3. **Data Extraction:** Accurately retrieve hidden text data from a steganographic image.

4. **Least Significant Bit (LSB) Technique:** Utilizes the LSB method for robust and efficient data hiding.

5. **Cross-Platform Compatibility:** The tool is written in Python, ensuring it runs on Windows, macOS, and Linux.

# **Getting Started 🚀**
To get a local copy of the project up and running, follow these simple steps.

 **Prerequisites**
Python 3.6 or higher

pip package manager

**Installation**
**Clone the repository:**

git clone https://github.com/RiyazShaik27/Steganography-GUI.git

Navigate to the project directory: Steganography-GUI

**Create and activate a virtual environment:**

# For Python projects
python -m venv venv
source venv/bin/activate  # On macOS/Linux
venv\Scripts\activate     # On Windows
**Install the required packages:**

pip install -r requirements.txt
Run the application/scripts:
python main.py
